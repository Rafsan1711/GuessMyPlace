# GuessMyPlace - একটি বুদ্ধিমত্তাসম্পন্ন জিওগ্রাফি গেসিং গেম

**GuessMyPlace** হলো একটি Akinator-স্টাইল ওয়েব অ্যাপ্লিকেশন যা আপনার মনে মনে ভাবা দেশ, শহর বা ঐতিহাসিক স্থানটি কিছু নির্দিষ্ট প্রশ্নের মাধ্যমে অনুমান করতে পারে। এটি অত্যন্ত আধুনিক ডিজাইন এবং দক্ষ অ্যালগরিদম ব্যবহার করে তৈরি করা হয়েছে।

## 🌍 মূল বৈশিষ্ট্যসমূহ

- **তিনটি ক্যাটাগরি:** আপনি দেশ (Country), শহর (City) অথবা ঐতিহাসিক স্থান (Historic Place) নিয়ে খেলতে পারেন।
- **ইন্টেলিজেন্ট অ্যালগরিদম:** এটি "Information Gain" অ্যালগরিদম ব্যবহার করে সবচেয়ে উপযুক্ত প্রশ্নটি নির্বাচন করে, যাতে দ্রুততম সময়ে সঠিক জায়গায় পৌঁছানো যায়।
- **রেসপন্সিভ ডিজাইন:** মোবাইল, ট্যাবলেট এবং ডেস্কটপ—সব ডিভাইসেই দারুণভাবে কাজ করে।
- **স্ট্যাটাস ট্র্যাকিং:** আপনার গেম খেলার সংখ্যা এবং অনুমান করার সাফল্যের হার (Accuracy) লোকাল স্টোরেজে সংরক্ষিত থাকে।
- **মসৃণ এনিমেশন:** গেমটিকে জীবন্ত করতে আধুনিক এনিমেশন এবং ট্রানজিশন ব্যবহার করা হয়েছে।

## 🛠 প্রযুক্তিগত কাঠামো (Tech Stack)

- **Frontend:** React 19
- **Styling:** Tailwind CSS (ডিজাইনের জন্য)
- **Icons:** Lucide React
- **Logic:** TypeScript
- **Deployment Ready:** ES6 মডিউল এবং আধুনিক ব্রাউজার সাপোর্ট।

## 📂 প্রকল্পের গঠন (Project Structure)

- `App.tsx`: গেমের মূল লজিক এবং স্ক্রিন কন্ট্রোলার।
- `data/`: গেমের প্রশ্ন এবং জায়গার ডাটাবেস।
- `components/`: গেমের বিভিন্ন স্ক্রিন (Welcome, Game, Guess, Result) এবং প্রোগ্রেস বার।
- `utils/`: ইনফরমেশন গেইন ক্যালকুলেশন এবং লোকাল স্টোরেজ ইউটিলিটি।
- `types.ts`: সমস্ত টাইপ এবং ইন্টারফেস ডেফিনিশন।

## 🚀 গেমটি কিভাবে কাজ করে?

১. **ক্যাটাগরি নির্বাচন:** প্রথমে দেশ, শহর বা ঐতিহাসিক স্থানের মধ্যে যেকোনো একটি বেছে নিন।
২. **অনুমান শুরু:** গেমটি আপনাকে কিছু হ্যাঁ/না প্রশ্ন করবে।
৩. **অ্যালগরিদম:** প্রতি উত্তরের পর "Information Gain" লজিক ব্যবহার করে সিস্টেমটি সম্ভাব্য জায়গার তালিকা থেকে অপ্রয়োজনীয় নামগুলো বাদ দেয়।
৪. **ফলাফল:** যখন সিস্টেম নিশ্চিত হয় (বা ২০টি প্রশ্ন শেষ হয়), তখন এটি তার চূড়ান্ত অনুমান দেখায়।
৫. **লার্নিং:** আপনি সঠিক বা ভুল বললে সেটি সিস্টেমের সাফল্যে প্রভাব ফেলে।

## 🔧 লোকাল ডেভেলপমেন্ট
